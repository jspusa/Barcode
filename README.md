# Barcode Generator

簡易的 EAN-13 條碼產生器，使用瀏覽器即可運作。

## 使用方式
1. 以瀏覽器開啟 `index.html`。
2. 輸入前 12 位數字後按下 **產生條碼**。
3. 生成的條碼可直接下載為 SVG 檔案。
4. 可額外下載 **4x6 標籤紙（10.2cm x 15.2cm）PDF**，版面自動以 **11mm x 35mm** 條碼尺寸重複排列，方便自行裁切。
5. 也可上傳自訂圖片檔（例如 11mm x 35mm 或其他尺寸），輸入寬高（mm）後，自動排版成 4x6 PDF。
   - 預設會先依原檔案比例與尺寸自動帶入寬高，不再固定預設 11x35。
   - 若要手動調整，勾選「啟用自訂尺寸」後可自行輸入，或一鍵套用 11x35。
6. 支援直接上傳 AI 檔並自動轉 PNG 後排版（需為 PDF 相容的 AI 檔）。
   - 會自動從 AI 檔中尋找內嵌 PDF 區段再轉檔，提升相容性。
7. 輸出的每個貼紙區塊都會加上黑色邊框，方便裁切。
   - 條碼內容會與邊框保留安全距離，避免數字貼齊邊框難裁切。
8. 排版區塊之間無間隔，且從紙張邊界開始緊密排列，方便一刀切並提高可排數量。

此專案以 [JsBarcode](https://github.com/lindell/JsBarcode) 為核心實作。
